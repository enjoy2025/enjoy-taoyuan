# Enjoy · Taoyuan

> 在世外桃源，享受秩序中的快乐  
> A joyful, calm and well-structured system for modern organizations

---

## 一、项目简介

**Enjoy** 是一个面向 **多租户、多角色、多场景** 的后台管理与协作平台。  
项目以 **“快乐（Enjoy）”** 为核心理念，希望在复杂的组织与权限体系中，依然让系统 **清晰、自然、低摩擦**。

**Taoyuan（桃源）** 是 Enjoy 的世界观与架构隐喻：

> 像世外桃源一样  
> 秩序存在，但不压迫  
> 角色分明，却不混乱  
> 各司其职，各得其所

---

## 二、核心设计目标

- 🎯 **先稳后强**：先构建清晰可运行的系统骨架，再逐步增强能力
- 🧩 **模块解耦**：业务、框架、启动完全分离
- 🏗️ **可演进架构**：Spring Boot 单体起步，天然支持未来 Spring Cloud
- 🏢 **多租户优先**：Tenant 是系统的一等公民
- 🔐 **安全可控**：RBAC + JWT 双 Token，不牺牲可控性
- 🧰 **框架能力平台化**：缓存、日志、多租户、安全全部抽象为 framework/starter
